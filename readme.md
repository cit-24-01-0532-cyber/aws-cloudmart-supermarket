# CloudMart - AWS S3 Static Website Hosting

This project demonstrates how to host a high-performance, scalable supermarket landing page using **Amazon S3**.

## Features
* **Static Hosting:** Fully hosted on AWS S3.
* **Serverless Architecture:** No servers to manage.
* **Cost-Efficient:** Pay-as-you-go pricing.

## AWS Technologies Used
* **Amazon S3:** Used for storing web assets (HTML/CSS).
* **Bucket Policies:** Configured for public read access.
* **Static Web Hosting:** Enabled for the index document.
